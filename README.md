# Opções Matemáticas em Java

![Java](https://img.shields.io/badge/Java-8-blue)
![Swing](https://img.shields.io/badge/Swing-GUI-green)

Este é um projeto Java que apresenta uma interface gráfica do usuário (GUI) com várias opções matemáticas e utilitárias. Ele foi desenvolvido como parte de um estudo sobre desenvolvimento de interfaces gráficas em Java usando a biblioteca Swing.

## Funcionalidades Principais:

- **Calculadora:** Realiza operações básicas como adição, subtração, multiplicação e divisão.
- **Verificar Par ou Ímpar:** Determina se um número digitado pelo usuário é par ou ímpar.
- **Verificar Número Primo:** Verifica se um número digitado pelo usuário é primo.
- **Gerar Triângulo:** Gera um triângulo com base no número fornecido pelo usuário, centralizando os números.
- **Gerar Números Aleatórios:** Gera uma quantidade específica de números aleatórios dentro de um intervalo definido pelo usuário.

## Tecnologias Utilizadas:

- ![Java](https://img.shields.io/badge/Java-8-blue): Linguagem de programação utilizada.
- ![Swing](https://img.shields.io/badge/Swing-GUI-green): Biblioteca para construção da interface gráfica.

## Como Usar:

1. Clone o repositório.
2. Abra o projeto em sua IDE preferida.
3. Compile e execute a classe `MenuGUI.java`.
4. Explore as diferentes opções matemáticas disponíveis na interface gráfica.
